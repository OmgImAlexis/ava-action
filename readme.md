# xo-action

> ❤️ JavaScript happiness style linter GitHub Action


## Usage
To use the action simply add the following lines to your `.github/workflows/main.yml`.

```yaml
- name: xo
  uses: stoe/xo-action@master
  env:
    GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
